# funbro 方同学
funbro 方同学，面向孤独老年群体的电视手机娱乐助理

* 具有记忆，专属于你的私人助理。基于文件系统的 claude code 智能内核
* 类似蔚来 NOMI 的拟人界面。
* 本地语音识别和语音合成，7*24不关机在旧安卓手机上运行，隐私不泄漏。OLED屏黑色界面，常亮不会导致烧屏。
* 基本的蓝牙智能音箱的功能。随时语音对话。
* 通过投屏到任意屏幕，接管安卓系统的电视。基于 moonlight 技术实现，具有极为广泛的适配性。解决小米电视老年人用不来，找不到免费片子的问题。方同学本身提供了类似tvbox的功能，自带资源检索，视频播放能力。
* 通过接收任意屏幕的投屏，通过蓝牙鼠标键盘协议，和你共用同一台手机。基于 airplay（iphone投屏到方同学） 或者 sunshine（所有其他平台投屏到方同学） 技术实现发送端。解决刷手机，玩游戏，孤独寂寞的问题。随时有个 claude code跟你一起看，一起玩。它还能帮你玩，教你玩。
* 方同学运行在家里的手机上，做为你的微信联系人添加你的微信。不在家的时候可以通过微信文字，语音或者视频联系家中的方同学。

## 为什么不直接在孤独老年人的主力手机上运行方同学？

为什么不直接把“方同学”装在老年人自己的主力手机上，而是要专门弄一台旧安卓手机常驻家里、常亮摆在那儿？这其实是整个设计里最核心、最有温度的“取舍”，正好解决了几大老年人真实痛点：

### 1. 主力手机对老年人来说是“命根子”，不能随便折腾
- 老年人的手机里只有三个东西最重要：微信、健康码、银行卡。
- 他们最怕的就是“手机突然变了”“点一下就花钱”“不小心装了流氓软件被骗”。
- 如果直接在主力手机上装一个复杂、常驻后台、占内存、黑屏界面的“方同学”，老人99%会慌，会觉得“手机坏了”“被病毒控制了”，然后整天重启、卸载、找儿女哭。
- 而方同学跑在一台专门的“旧手机”上，老人一眼就知道：“哦，这个是家里新买的小电视/小音箱”，心理上完全不慌，这是“新增家电”，而不是“主力手机被改了”。

### 2. 常亮 + 大字体 + 黑色OLED界面，只有专用机敢这么干
- 老年人视力差，需要屏幕常亮、字体超大、界面极简。
- 主力手机敢常亮吗？不敢。一常亮就烧屏 + 耗电飞快 + 微信电话进不来。
- 专用旧手机可以用一块残血OLED屏（反正已经不值钱了），黑底白字常亮一个月也就掉1%寿命，老人随时抬头就能看到方同学在冲他笑、跟他说话，真正的“陪伴感”。

### 3. 投屏出去（手机 → 方同学 → 电视）：解决“老年人用不好电视盒子”的终极难题
- 老年人想看电视，理论上有小米电视、当贝盒子、电信机顶盒……但实际操作全是灾难：
  - 遥控器按键太多，找片子要进7级菜单
  - 广告多、会员多、免费片源找不到
  - 儿女走了就彻底不会用了
- 方同学的玩法是：儿女用自己手机投屏到家里那台“方同学手机”，方同学再投到电视上。
  - 儿女可以远程帮爸妈放《长津湖》《甄嬛传》第73集精确进度
  - 方同学自己也带片源搜索，直接语音说“方同学，放个抗战片”，秒出片单
- 老人完全不用学电视遥控器，只需要对家里那张“熟悉的脸”说话就行。

### 4. 接收投屏（手机/电脑 → 方同学）：解决“一个人打游戏、刷视频太孤单”
- 老人其实也刷抖音、玩消消乐、看直播，但是一个人刷没意思。
- 这时候把手机投屏到方同学（大屏幕），方同学就能陪着看、陪着聊、陪着玩，甚至帮他自动消除三行、帮他斗地主出牌、帮他识别直播间骗局。
- 相当于把“儿女陪看陪玩”这个最珍贵的陪伴，变成了7×24可随时唤醒的服务。

### 5. 微信三模通信：解决“儿女不在家也想找人说话”
- 方同学是微信联系人，老人可以：
  - 发语音（家里方同学立刻回答）
  - 打视频（看到家里的实时画面 + 方同学的脸）
  - 发文字（方同学用大字读出来）
- 这其实把“家里那台旧手机”变成了一个永不离家的“分身儿女”。

总结一句话：
把方同学单独跑在一台专用旧手机上，而不是装在老人主力手机里，就是为了把“最复杂、最重度的陪伴功能”从“最脆弱、最不能出错的主力手机”上剥离出来，让老人既用得上最强的AI陪伴，又完全不打扰、不吓到、不折腾他们原来仅有的那点数字安全感。

这才是真正懂老人的产品逻辑：  
不是把最牛的技术塞进他们的主力手机，  
而是再给家里添一个“永不走的儿女”。
