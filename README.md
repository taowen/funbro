# funbro 方同学
funbro 方同学，面向孤独老年群体的电视手机娱乐助理

* 具有记忆，专属于你的私人助理。基于文件系统的 claude code 智能内核
* 类似蔚来 NOMI 的拟人界面。
* 本地语音识别和语音合成，7*24不关机在旧安卓手机上运行，隐私不泄漏。OLED屏黑色界面，常亮不会导致烧屏。
* 基本的蓝牙智能音箱的功能。随时语音对话。
* 通过投屏到任意屏幕，接管安卓系统的电视。基于 moonlight 技术实现，具有极为广泛的适配性。解决小米电视老年人用不来，找不到免费片子的问题。方同学本身提供了类似tvbox的功能，自带资源检索，视频播放能力。
* 通过接收任意屏幕的投屏，通过蓝牙鼠标键盘协议，和你共用同一台手机。基于 airplay（iphone投屏到方同学） 或者 sunshine（所有其他平台投屏到方同学） 技术实现发送端。解决刷手机，玩游戏，孤独寂寞的问题。随时有个 claude code跟你一起看，一起玩。它还能帮你玩，教你玩。
* 方同学运行在家里的手机上，做为你的微信联系人添加你的微信。不在家的时候可以通过微信文字，语音或者视频联系家中的方同学。

## 为什么不直接在孤独老年人的主力手机上运行方同学？

为什么不直接把“方同学”装在老年人自己平时用的主力手机上，而是要额外用一台旧安卓手机常年插电放在家里当专属设备？

以下是几个非常现实、老年人实际使用场景里几乎无法绕过去的核心原因：

1. **老年人主力手机几乎不可能7×24小时开机且插着电不拿走**  
   - 老年人出门买菜、跳广场舞、走亲戚、住院、住养老院……手机必须随身带走。
   - 你不可能要求一个70-80岁的老人在家也把手机一直插着电、常亮、放在客厅当“智能音箱”。
   - 结果就是：你一出门，方同学就“失踪”了，家里彻底没人应。

2. **老年人对手机的“独占欲”极强，不接受后台被长期占用的服务**  
   - 大部分老人只要一看到“后台有程序在跑、电量掉得快、存储被占”，第一反应就是去关掉、卸载。
   - 即使你做好了省电和隐藏，他们也会因为“手机突然说话”“不知道哪来的声音”而慌张，然后长按关掉你的app，或者直接恢复出厂设置（真事非常多）。

3. **主力手机性能太差 + 系统版本太老 + 各种管家乱杀后台**  
   - 孤寡老人的手机基本都是几百块的4+64G低端机，系统停在安卓10/11，各种国产rom+老人模式+手机管家，任何本地大模型/语音识别都会被杀得干干净净。
   - 你就是做到了完全本地跑，也跑不流畅，还卡，还发烫，他们一样会嫌弃、卸载。

4. **需要常亮屏幕做“拟人化伴侣界面”**（像蔚来NOMI那种小脑袋）  
   - 主力手机不可能允许你常亮屏幕（烧屏先不说，老年人看到电量狂掉会抓狂）。
   - 而专门放一台旧手机，用OLED黑底界面，常亮就是放着当电子相框也没人管，这才是真正可落地的形态。

5. **需要接管电视、做投屏接收端、当蓝牙音箱**  
   - 这些功能都需要设备一直通电、联网、开着Wi-Fi/蓝牙、插着HDMI或无线投屏。
   - 主力手机做不到“一直插着电放在电视柜上”，它得跟着人走。

6. **微信远程联系“家里的他”才有情感意义**  
   - 老人在外地医院、子女家住几天，最想的就是“家里还有个‘人’陪着我说话”。
   - 如果方同学就是他自己兜里的手机，那拨微信视频等于自己给自己打视频，情感陪伴感瞬间归零。
   - 必须是“家里还有另一个‘他’在等我”，才治孤独。

总结一句话：
老年人真正的孤独不是“手机里没人陪”，而是“家里没人应”。  
方同学必须是一个“放在家里、永不离家、永不关机、会有声音会亮灯”的实体存在，才真正能填补这个空缺。

所以必须单独用一台旧手机插电放在家里，这不是技术妥协，而是精准击中了“家有老小”的情感刚需。  
这才是funbro方同学能真正打动孤寡老人的核心产品逻辑。
